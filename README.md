# podcast-transcriber-jp

日本語ポッドキャストの「話者ごとに録音された音声トラック」をまとめて文字起こしする Windows 用ツールです。

ローカルで動きます。音声はインターネットには送信されません。

## はじめての方へ（インストール手順）

1. [Releases](../../releases) から **podcast-transcriber-jp.zip** をダウンロード
2. ダウンロードした ZIP を右クリック → **すべて展開** で解凍
3. 解凍後のフォルダの中にある **`start.bat`** をダブルクリック

これだけです。Python のインストールなど事前準備は不要です。

NVIDIA GeForce などの GPU が PC に載っている場合は自動的に GPU が使われ、10倍ほど高速になります。GPU がない場合は CPU で動作します（時間はかかりますが結果は同じです）。

### 「Windows によって PC が保護されました」と出たら

Microsoft が署名していない実行ファイルなので警告が出ます。

1. 青い画面の **「詳細情報」** をクリック
2. 下に出てきた **「実行」** をクリック

## 初回起動について

初回起動時のみ、文字起こしモデル（約 **3GB**）が自動でダウンロードされます。ネット環境にもよりますが **5〜15分** かかります。

ダウンロード先は ZIP 配下の `models/hf_cache/` です。2回目以降は再ダウンロード不要です。

## 使い方

1. `start.bat` を起動 → GUI ウィンドウが開く
2. **話者別トラック** に、話者ごとに録音された wav/mp3 ファイルを **ドラッグ&ドロップ** で2つ以上追加
3. **出力先** が自動で決まる（変更可）
4. （任意）**用語リスト**に固有名詞を1行ずつ書く（精度が上がります）
5. **実行** ボタンを押す

72分の音声の処理時間目安:
- GPU 搭載PC: 30分〜1時間
- CPU のみ: 5〜10時間

## 出力されるファイル

出力フォルダに以下が作成されます:

- `transcript_speaker_tracks.json` — 文字起こし本体（構造化データ）
- `transcript_speaker_tracks.clean.md` — 読みやすい本文
- `transcript_speaker_tracks.review.md` — 注釈つき詳細版
- `transcript_speaker_tracks.simple.txt` — シンプルなテキスト
- `transcript_speaker_tracks.log.txt` — 処理ログ

## 動作するPC

- **OS**: Windows 10 / 11（64bit）
- **メモリ**: 8GB 以上（16GB 推奨）
- **空き容量**: 5GB 以上
- **GPU**（任意）: NVIDIA GeForce などがあると自動で利用されます（CUDA 12.x 対応）

## トラブルシュート

**Q. 起動しない / 黒い画面が一瞬出て消える**
- ZIP を「すべて展開」ではなく、ZIP のままダブルクリックしていませんか? 必ず一度解凍してから `start.bat` を実行してください

**Q. 初回起動で「ダウンロード失敗」と出る**
- インターネット接続を確認してください。社内プロキシ環境では使えないことがあります
- 一度すべてのウィンドウを閉じて、再度 `start.bat` を実行してください

**Q. メモリ不足エラーになる**
- 他のアプリを閉じてください
- 1時間以上の長い音声は、短く分割してから処理することを検討してください
